# Banking Operations Widget

## 📝 Описание проекта

Виджет для обработки и анализа банковских операций клиента. Проект предоставляет инструменты для фильтрации и сортировки транзакций, что позволяет эффективно управлять банковскими данными.

### Основные возможности

- **Фильтрация транзакций** по статусу выполнения
- **Сортировка транзакций** по дате
- **Типизированные функции** с полной аннотацией типов
- **Валидация входных данных**

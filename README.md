# Проект обработки банковских транзакций

##  Описание проекта

Данный проект представляет собой набор утилит для обработки данных банковских транзакций. Модуль `processing` содержит функции для фильтрации и сортировки списков транзакций, представленных в виде словарей.

Проект разработан в рамках изучения работы с данными в Python, включая:
- Фильтрацию данных по заданным критериям
- Сортировку данных по дате
- Обработку списков словарей
- Написание чистого и документированного кода

##  Функциональность

Модуль `processing` предоставляет следующие функции:

### 1. `filter_by_state(data, state='EXECUTED')`

Фильтрует список транзакций по статусу.

**Параметры:**
- `data` (list): Список словарей с данными транзакций
- `state` (str): Статус для фильтрации (по умолчанию `'EXECUTED'`)

**Возвращает:** Новый список словарей с указанным статусом

### 2. `sort_by_date(data, descending=True)`

Сортирует список транзакций по дате.

**Параметры:**
- `data` (list): Список словарей с данными транзакций
- `descending` (bool): Порядок сортировки:
  - `True` - убывание (новые сначала)
  - `False` - возрастание (старые сначала)

**Возвращает:** Новый отсортированный список словарей
